# 🚌 CloudBus India - Animated Reservation System

CloudBus India is a lightweight, high-performance single-page application (SPA) that simulates an end-to-end Indian interstate luxury bus ticketing system. Built using HTML, CSS, and vanilla JavaScript, it delivers smooth UI animations, interactive seat selection, and a real-time booking experience.

## ✨ Features

- 🚌 Route search system (From / To cities)
- 💺 Interactive seat selection (2+1 sleeper layout)
- 💰 Dynamic fare calculation
- 👤 Passenger details form validation
- 💳 Payment simulation (UPI mock system)
- 🎫 Auto-generated PNR ticket system
- 📋 "My Bookings" history section
- 🎨 Modern glassmorphism UI design
- ⚡ Fully responsive single-page app
